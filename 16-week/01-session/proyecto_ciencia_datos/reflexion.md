# Reflexión del Proyecto- Visualización de Datos

El desarrollo del tablero permitió comprender la importancia de convertir datos crudos
en información útil. La visualización facilita identificar patrones, comparar regiones
y reconocer productos con mayor movimiento.

El uso de Plotly permite una experiencia interactiva que mejora el análisis,
mientras que las métricas clave ayudan a tomar decisiones informadas
sobre el rendimiento comercial.

Este proyecto demuestra que un dashboard bien estructurado no solo comunica datos,
sino que orienta estrategias y acciones empresariales.
